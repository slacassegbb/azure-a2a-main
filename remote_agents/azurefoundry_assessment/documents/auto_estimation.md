# Auto Damage Assessment & Estimation

_Last updated: 2025-09-24 | Version 3.0_

---

## 1. Damage Categories
- **Collision:** Crumpled panels, bent frame, deployed airbags.  
- **Comprehensive:** Fire, theft recovery, hail dents, broken glass.  
- **Flood:** Interior waterline, corrosion, hydrolocked engine.  

---

## 2. Cost Components
- **Labor:** $70–$95/hr (rural), $90–$125/hr (urban).  
- **Paint/Materials:** $40–$60/hr.  
- **Towing/Storage:** $150 tow, $25–$40/day storage.  
- **Diagnostic fees:** $150–$300.

## 4. Severity Scale
- **Minor:** <$1,500, cosmetic scratches, one panel.  
- **Moderate:** $2k–$8k, multiple panels, airbag deployment.  
- **Severe:** >$10k, frame/fire/flood, or repair ≥75% ACV (total loss).  

---

## 5. Examples
- **Front bumper + headlight:** $2,250.  
- **Hailstorm (40 dents, hood + roof respray):** $5,400.  
- **Flood above dashboard:** $21,500 → Total loss if ACV < $25k.  
- **Rear-end crash (bumper, trunk, 2 airbags):** $9,800.  
- **Theft recovery (missing wheels, interior damage):** $7,200.
