# Health Cost Estimation

_Last updated: 2025-09-24 | Version 3.0_

---

## 1. Services
| Service | Cost Range |
|---------|------------|
| Hospital stay/day | $2,500–$5,000 |
| ER visit | $800–$2,000 |
| X-ray | $150–$300 |
| MRI | $1,200–$3,000 |
| CT scan | $1,000–$2,500 |
| Outpatient surgery | $5,000–$15,000 |
| Childbirth + 2 days | $12,000–$18,000 |
| Mental health session | $150–$250 |
| Specialty drug/month | $2,500–$5,000 |

---

## 2. Bundled Examples
- **ER + stitches + X-ray:** $1,700.  
- **Maternity delivery:** $17,500.  
- **MRI + specialist:** $2,400.  
- **Appendectomy (3-day stay):** $28,000.
