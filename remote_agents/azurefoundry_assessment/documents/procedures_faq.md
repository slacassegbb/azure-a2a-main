# Estimation Procedures & FAQs

_Last updated: 2025-09-24 | Version 3.0_

---

## 1. Workflow
1. Assignment → File received.  
2. Review → Policy notes, photos.  
3. Inspection → On-site or virtual.  
4. Estimation → Apply tables, labor, depreciation.  
5. Report → Structured summary.  
6. Handoff → Estimation packet delivered to claims team.  

---

## 2. Escalation
- >$50k → Senior estimator sign-off.  
- Unsafe site → Notify safety/compliance.  
- Missing info → Request reinspection.  

---

## 3. QA Checklist
- Clear photos, labeled by location/item.  
- Labor hours match industry averages.  
- Materials priced from current guides.  
- Hidden damage considered.  

---

## 4. FAQs
- **Do estimators decide coverage?** No, only costs.  
- **How to handle luxury/custom items?** Use comparable pricing, flag for adjuster.  
- **What about surge pricing post-disaster?** Apply +25–50%.  
- **Timeline for reports?** Within 48 hours.  
- **Can estimates be ranges?** Yes, provide low–high when uncertainty exists.
