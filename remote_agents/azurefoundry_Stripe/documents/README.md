# Stripe Agent Documents

This folder is for optional documents that can be uploaded to the agent's vector store for file search capabilities.

## Supported File Types
- `.txt` - Plain text files
- `.md` - Markdown files
- `.pdf` - PDF documents
- `.docx` - Word documents
- `.json` - JSON files
- `.csv` - CSV files

## Usage
Place any Stripe-related documentation here:
- Stripe API documentation
- Company-specific billing policies
- Custom pricing guides
- Integration documentation

The agent will automatically upload these files and create a vector store for semantic search.
